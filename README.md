# 포켓몬 도감

포켓몬 api 를 사용해서  pwa 웹사이트로 만들어본 포켓몬 도감 프로젝트입니다

## 사용한 기술

- next.js
- graphql
- apollo/client
- pwa
- typescript
- recoil
